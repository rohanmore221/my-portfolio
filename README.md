# Modern Portfolio Website

A sleek, responsive, and fully animated portfolio website. Built with pure HTML, CSS, and JavaScript. Zero dependencies, extremely fast, and incredibly easy to deploy.

## Features
- **Modern UI:** Glassmorphism, CSS Variables, clean typography.
- **Dark Mode Default:** Premium deep-dark aesthetics.
- **Scroll Animations:** Built using `IntersectionObserver` for seamless performance.
- **Fully Responsive:** Adapts perfectly to mobile and desktop screens.
- **Zero Build Step:** 100% static, edit and push right away.

## How to Deploy on GitHub Pages

Since this website is completely static (no build frameworks like React or Next.js), deploying it to GitHub is instant and natively supported by GitHub Pages.

**Step-by-step Instructions:**

1. **Create a new repository** on GitHub (e.g., `my-portfolio`). Make it public.
2. **Upload these files** to your repository:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md` (Optional)
   *(You can simply drag and drop them into the GitHub web interface or use git commands).*
3. **Commit** your changes.
4. Go to the repository's **Settings** tab.
5. In the left sidebar, click on **Pages**.
6. Under **Build and deployment**, ensure the **Source** is set to `Deploy from a branch`.
7. Under **Branch**, select `main` (or `master`), choose the `/ (root)` folder, and click **Save**.
8. Wait a minute or two, and refresh the page. You will see a green banner that says *"Your site is live at https://[yourusername].github.io/my-portfolio/"*.

Congratulations! Your modern portfolio is now live on the internet.
